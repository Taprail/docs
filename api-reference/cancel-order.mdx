---
title: "Cancel order"
api: "POST https://api.taprail.app/v1/merchant/orders/{id}/cancel"
description: "Void an unpaid order so it can no longer be paid."
---

Cancels an order that hasn't been paid yet — status `created` or `awaiting_user`. Paid,
declined, expired, or refunded orders can't be cancelled (refund an approved order instead).

<ParamField path="id" type="uuid" required>The order id.</ParamField>

No request body.

### Response

<ResponseField name="status" type="string">`cancelled`.</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.taprail.app/v1/merchant/orders/550e8400-.../cancel \
  -H "Authorization: Bearer tr_test_..."
```
</RequestExample>

<ResponseExample>
```json 200
{ "status": "cancelled" }
```
```json 400
{
  "error": "bad_request",
  "message": "order can no longer be cancelled"
}
```
</ResponseExample>

<Note>
  Unpaid orders also expire on their own 30 minutes after creation — cancel is for when you
  want them gone immediately (e.g. the customer edited their cart).
</Note>
